Typography

Documentation and examples for brand typography, including standards for print as well as web and email.

Print

The primary typeface for all Dana-Farber communications, Univers, is made up of five weights: Light, Roman, Bold, and Black. For each weight there is also an Oblique version. The majority of Dana-Farber’s communications should be set in Univers Roman, with light and heavier weights serving as heads, subheads, or in creative typographic applications.

Web and Email

Univers, the primary typeface for all Dana-Farber communications, is not a default font and not viewable on most devices and browsers. However, it can be installed on a webserver and used for the main website and digital applications. If you do not have access to Univers and want to create something for email or web, we suggest using websafe Arial as your alternate font choice.
